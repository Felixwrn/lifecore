WRN LifeCore Docs

The next-generation modular life system for Minecraft servers powered by WRN API.

Installation

  1. Download WRN-API.jar
  2. Download LifeCore.jar
  3. Put both files into your plugins folder
  4. Restart your server

Commands

/livesgui
/langgui
/modegui
/market
/wrn devmode
/wrn confirm <plugin>

Configuration

default-lives: 3

dev-mode:
  enabled: true
  confirmed-plugins: []

Developer API

WRNAPI.getDevMode().isEnabled();

<dependency>
    <groupId>de.wrn</groupId>
    <artifactId>wrn-api</artifactId>
    <version>1.0</version>
</dependency>

DevMode

DevMode allows external plugins to access advanced WRN systems safely. 

/wrn devmode
/wrn confirm ExamplePlugin
